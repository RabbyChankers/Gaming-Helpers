# Eldritch-Encounter-Generator

A lightweight Python utility for Tabletop RPG Game Masters to generate atmospheric, Lovecraftian descriptions on the fly.

## Features
*   Generates randomized "Cthulhu Mythos" style descriptions.
*   Uses a modular Class-based structure for easy expansion.
*   Zero external dependencies (Standard Library only).

## How to Run
1. Ensure you have Python 3.x installed.
2. Clone this repository.
3. Run the script via terminal:
   ```bash
   python generator.py
