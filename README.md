# RPG Generator Toolkit

A lightweight Python toolkit containing simple random generators for tabletop roleplaying games.

## Included Scripts

### 1. Lovecraftian Description Generator
Generates atmospheric eldritch descriptions for horror and investigative RPG sessions.

### 2. D&D 5e NPC Generator
Generates random NPC names and short descriptions for fantasy campaigns.

## Features
- Written in Python 3
- Simple class-based design
- Easy to expand with more tables and traits
- No external dependencies

## How to Run

Run either script from the terminal:

```bash
python generator.py
python npc_generator.py
